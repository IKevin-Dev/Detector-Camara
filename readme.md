# Detector de Gestos con Cámara

Detector de gestos basicos en tiempo real usando Python, OpenCV y MediaPipe.

## Gestos detectados
- ✊ Puño
- 🖐️ Mano abierta
- ☝️ Apuntando
- ✌️ Paz / Victoria
- 👍 Pulgar arriba

## Instalación

1. Clona el repositorio
   git clone https://github.com/tu-usuario/detector-camara.git
   cd detector-camara

2. Crea el entorno virtual
   python -m venv venv
   venv\Scripts\activate

3. Instala las dependencias
   pip install -r requirements.txt

4. Ejecuta
   py detector_gestos.py (Gestos manuales)
   py detector.py (Solo rostros)

## Requisitos
- Python 3.12+
- Cámara web
